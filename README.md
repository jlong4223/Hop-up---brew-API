# Hop up & Brew with Buds

A simple web application that displays beer information related to the beer that a user inputs. A random beer can also be generated by the click of a button.

## Technologies Used 

- HTML5
- CSS3
- Javascript
- jQuery
- Google Fonts
- Beer API
